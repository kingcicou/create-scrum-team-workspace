# Product Backlog 说明

Product Backlog 是唯一的需求来源，由 PO 负责排序与维护。

## 条目类型

| 类型 | 说明 |
| --- | --- |
| Epic | 跨多个 Sprint 的大型需求 |
| Story | 最小可交付价值单元 |
| Bug | 缺陷修复 |
| Tech-Debt | 技术债治理 |
| Spike | 技术探针 |

## 状态流

New → Ready for Refinement → Ready for Sprint → Selected for Sprint → In Progress → Done
