# 路线图与北极星指标

> 路线图回答"接下来 3 个月做什么"，北极星指标回答"我们是否在变好"。
> 不要把路线图写成承诺时间表——它是基于当前假设的最佳预期，会随 Sprint 验证持续调整。

## 1. 北极星指标（North Star）

| 指标名 | 定义 | 当前基线 | 12 周目标 | 度量方式 | 责任人 |
| --- | --- | --- | --- | --- | --- |
|  |  |  |  |  |  |

**选指标原则**：

- 体现**核心价值兑现**（不是产出量，是用户得到的价值）
- 团队可影响（不要选不可控的宏观指标）
- 高频可测（至少每 Sprint 看一次）
- **单一**或顶多 2 个，避免"什么都重要 = 什么都不重要"

> 任何路线图调整必须记录原因（通常来自 `02_价值假设` 验证结果或 `03_竞品与市场` 新信号）。

## 5. 反模式

- 把路线图当对外承诺（应明示"基于当前假设，会调整"）
- 北极星指标选成"DAU"或"PV"等无法体现价值兑现的虚指标
- 二级指标过多导致团队失焦
- 路线图调整无原因记录，下个 Sprint 不知道为什么改了
