## 字段说明

| 字段 | 何时填 | 谁填 |
| --- | --- | --- |
| ID | 新建产出时 | 产出人 / SM |
| 优先级 | 新建产出时 | 产出人 / PO / SM |
| 产出物名称 | 新建产出时 | 产出人 |
| 存放位置 | 产出完成时，以真实路径为准 | 产出人 |
| 主责人 | 新建产出时 | 产出人 / SM |
| 状态 | 状态变化时 | 产出人 / 领取人 |
| 产出时间 | 产出完成时 | 产出人 |
| 依赖项 | 新建或发现依赖时 | 产出人 |
| 领取人 | 产物被消费时 | 消费者 |
| 参考人 | 新建产出时 | 产出人 |
| 交付时间 | 交付给下一角色时 | 产出人 |
| 变更摘要 | 每次实质性修改 | 修改人 |
| 备注 | 有风险、阻塞或特殊说明时 | 任何人 |
| 文档版本 | 文档初稿/修订/归档时 | 产出人 |
| 评审状态 | 文档或规范类产出 | 产出人 / 评审人 |
| 评审人 | 提交评审时 | 产出人指定，评审人确认 |
